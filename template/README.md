# Terra Draw Workshop Template (FOSS4G 2026)

This is the local development template for the
[Terra Draw workshop](https://github.com/watergis/terradraw-workshop).
It is a minimal SvelteKit + TypeScript project with a MapLibre GL JS map of
Hiroshima and an empty sidebar, ready for the workshop exercises.

## Getting started

```bash
pnpm install
pnpm dev
```

Then open http://localhost:5173 and follow the workshop documentation. We
mainly edit `src/routes/+page.svelte` during the workshop.

## Installed drawing libraries

- [terra-draw](https://github.com/JamesLMilner/terra-draw)
- [terra-draw-maplibre-gl-adapter](https://github.com/JamesLMilner/terra-draw)
- [@watergis/maplibre-gl-terradraw](https://github.com/watergis/maplibre-gl-terradraw)
