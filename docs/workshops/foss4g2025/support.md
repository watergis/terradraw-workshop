If you found any bugs or you have any idea to improve TerraDraw or the plugin, please create any issues in the repo.

## Contribution to TerraDraw community

Your support to TerraDraw is necessary to grow the community. If you like TerraDraw library, please star or fork repositories.

Please also feel free to make a pull request any time to contribute the projects.
