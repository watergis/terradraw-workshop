# Advanced features of Terra Draw

In this section, you'll learn about Terra Draw's advanced features with 4 exercises.

## Custom styling

You will learn how to change styles such as colors, line width, opacity and
dash patterns for each Terra Draw mode.

## Event handling

You will learn how to handle Terra Draw events in this exercise.

## Data management

You will learn how to manage data (loading data, getting and removing data) in Terra Draw.

## Undo / Redo

You will learn how to enable Terra Draw's new history feature (added in
v1.26) and wire it to Undo / Redo buttons and keyboard shortcuts.
